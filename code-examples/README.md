# Code Examples Library

This folder contains minimal, focused snippets used across the 12-week LeafGuard AI project. These are not full applications; they are small reference examples you can adapt into your weekly build tasks.

## Structure

| Area | Folder | Purpose |
|---|---|---|
| Android basics | `android/` | Activities, XML layouts, intents, permissions |
| Backend | `backend/` | FastAPI prediction endpoint and test client |
| ML | `ml/` | Preprocessing, mock classifier, TFLite conversion outline |
| Database | `database/` | Room entity, DAO, database, repository pattern |
| XML | `xml/` | Disease library XML and parser pattern |
| Networking | `networking/` | Retrofit API interface and upload request |
| Notifications | `notifications/` | Notification channel and reminder notification |
| Testing | `testing/` | Unit-test and manual-test examples |
| Final | `final-submission/` | Release checklist and demo script helper |

## Learning rule

Do not paste a snippet before you can answer:

1. What problem does this snippet solve?
2. Which week uses it?
3. Which file in my project should receive it?
4. What input does it require?
5. What output or side effect should I see?
6. What failure cases must I handle?
