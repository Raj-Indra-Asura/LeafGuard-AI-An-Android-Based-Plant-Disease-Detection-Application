## Viva opening statement

LeafGuard AI is an Android-based plant disease detection application. It demonstrates Android UI, camera and gallery input, API communication, local storage, XML parsing, notifications, testing, and ML integration through cloud and offline inference.

## Do not claim

- Do not claim medical/agricultural certainty.
- Do not claim the model is trained by you unless you actually trained it.
- Do not claim 100% accuracy.
- Do not claim production deployment unless deployed and tested.
