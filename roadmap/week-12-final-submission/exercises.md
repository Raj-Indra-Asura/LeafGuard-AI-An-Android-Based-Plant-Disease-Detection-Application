## APK Building

### Generate Signed APK

1. Build → Generate Signed Bundle/APK
2. Choose APK
3. Create new keystore (or use existing)
4. Build release APK
5. Test installation on physical device

### APK Testing Checklist

- [ ] Installs successfully
- [ ] All permissions requested
- [ ] Camera works
- [ ] Predictions work (cloud and offline)
- [ ] Database operations work
- [ ] No crashes in normal use
- [ ] Icons and branding correct

## Presentation Tips

1. **Practice demo** 5+ times before recording
2. **Script key talking points** - don't memorize, but have notes
3. **Show code briefly** - just key parts, not entire files
4. **Emphasize CSE 2206 topics** - mention syllabus explicitly
5. **Be honest about limitations** - acknowledge what doesn't work perfectly
