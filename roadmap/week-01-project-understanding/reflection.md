## Section 1: Learning & Understanding

### Prompt 1: Conceptual Breakthroughs

**Question:** What concept from Week 01 was initially confusing but now makes sense? Describe your "aha!" moment.

**Guidance:**
- Think about three-tier architecture, MVVM pattern, data flow, etc.
- What was unclear at first?
- What helped you understand it? (diagram, example, explanation)
- How would you explain it to someone else now?

**Example Response:**
```
The MVVM pattern was confusing initially. I didn't understand why we need three separate components (Model, View, ViewModel) when Activity alone could handle everything. My "aha" moment came when I learned about screen rotation—Activities get destroyed, but ViewModels survive. This means network calls in ViewModel won't be lost when user rotates the phone. Now I understand MVVM prevents bugs and makes code testable. I would explain it as: "ViewModel is a safe place to put logic that should survive Activity death."
```
