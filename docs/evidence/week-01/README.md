---

**Tips for Evidence Collection:**
- Take screenshots at each step
- Document all meetings with date and details
- Save all versions of documents (use version control)
- Keep backup copies in cloud storage
- Organize files with clear naming convention
- Add timestamps to all evidence files

**Naming Convention:**
- Use format: `YYYYMMDD_description.ext`
- Example: `20260525_guide_meeting_minutes.md`
- Example: `20260525_android_studio_setup.png`
