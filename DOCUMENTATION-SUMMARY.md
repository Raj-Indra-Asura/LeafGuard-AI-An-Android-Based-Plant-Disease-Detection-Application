1. Follow weekly evidence collection (week-03 through week-10)
2. Take screenshots at each milestone
3. Document code as you write
4. Keep meeting notes updated
5. Track progress against `final-definition-of-done.md`

### Phase 3: Testing (Week 11)
1. Execute test cases from `test-cases-template.md`
2. Document all test results
3. Fix bugs and retest
4. Collect testing evidence

1. Fill in `final-report-template.md`

1. Study `viva-questions.md` thoroughly
